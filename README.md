
A small desktop game built with [Electron](https://www.electronjs.org/) that I developed as a fun project to learn desktop app development and JavaScript.

## Overview

This game is a lightweight, interactive desktop application that demonstrates basic game mechanics using HTML, CSS, and JavaScript inside an Electron environment.  

- Built with **Electron**  
- Cross-platform desktop app (Windows, macOS, Linux)  
- Fun, minimalistic gameplay with simple controls  

## Features

- Playable on your desktop without a web browser  
- Responsive and lightweight  
- Easy to modify and expand for learning purposes
